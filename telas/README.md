# Telas implementadas

Esta pasta deve conter as quatro telas referente as entrega dos protótipos de tela.

Indique quais telas você escolheu e faça um breve descrição de cada uma delas. 

- [Tela de login](#) 
  - Interface de autenticação onde o usuário insere suas credenciais para acessar as funcionalidades restritas do sistema. Garante a segurança e o controle de acesso com base no nível de permissão de cada perfil. 

- [Tela de cardápio](#) 
  - Catálogo digital que exibe os produtos disponíveis na lanchonete. Conta com recursos de categorização e filtros para facilitar a navegação, permitindo que o cliente visualize detalhes dos itens e inicie o processo de pedido.

- [Tela de cadastro de Usuario](#) 
  - Formulário de registro responsável pela coleta e validação de dados de novos usuários. Permite a criação de uma conta no sistema, viabilizando futuras interações e o acesso personalizado à plataforma.

- [Tela de gestão de pedidos](#) 
  - Painel de controle restrito a administradores e funcionários. Permite o acompanhamento e a atualização do fluxo de trabalho, organizando os pedidos por status (Novos, Em Preparo e Prontos) para garantir um gerenciamento ágil da operação.
